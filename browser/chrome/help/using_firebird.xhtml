Χρησιμοποιώντας τον &brandFullName;

Καλώς ήλθατε στον &brandFullName;! Ο &brandShortName; είναι ένας περιηγητής διαδικτύου που μπορείτε να χρησιμοποιήσετε για ανάγνωση σελίδων και αναζήτηση στο διαδίκτυο.

Σε αυτή την ενότητα:
  • Περιήγηση ιστοσελίδων
  • Αναζήτηση
  • Αντιγραφή, Αποθήκευση και Εκτύπωση σελίδων
  • Βελτιώνοντας την ταχύτητα και την σποτελεσματικότητα

Περιήγηση ιστοσελίδων

Προβολή της αρχικής σας σελίδας

Όταν ξεκινάει ο &brandShortName;, βλέπετε την αρχική σας σελίδα. Η προεπιλογή είναι η αρχική σελίδα του &brandShortName;

Συμβουλές:

  • Για να εμφανίσετε περισσότερο περιεχόμενο στην οθόνη σας, μπορείτε να χρησιμοποιήσετε τη λειτουργία Πλήρους οθόνης. Η λειτουργία Πλήρους οθόνης συμπυκνώνει τις εργαλειοθήκες του &brandShortName; σε μια μικρή εργαλειοθήκη. Για να ενεργοποιήσετε τη λειτουργία Πλήρους οθόνης, απλά πατήστε Προβολή > Πλήρης οθόνη ή πατήστε F11.
  • Για να μεταβείτε στην αρχική σελίδα σας γρήγορα, πατήστε &altKey;+Home.

Μετακίνηση σε άλλη σελίδα

Μπορείτε να μεταβείτε σε μια νέα ιστοσελίδα πληκτρολογώντας τη διεύθυνση της ή το URL στη γραμμή διευθύνσεων. Τα URL συνήθως ξεκινάνε με "http://" και συνεχίζουν με ένα ή περισσότερα ονόματα που χαρακτηρίζουν τη σελίδα. Ένα παράδειγμα είναι το "http://www.mozilla.org/".

  1. Κάντε κλικ στη γραμμή διευθύνσεων για να επιλέξετε το URL που είναι ήδη εκεί.
  2. Πληκτρολογήστε το URL της σελίδας που θέλετε να επισκεφθείτε. Το URL που πληκτρολογείτε αντικαθιστά ότι κείμενο υπάρχει στη γραμμή διευθύνσεων.
  3. Πατήστε &enterKey;.

Συμβουλή: Για να επιλέξετε γρήγορα το URL στη γραμμή διευθύνσεων, πατήστε &accelKey;+L.

Δεν ξέρετε το URL; Προσπαθήστε να πληκτρολογήσετε κάτι σχετικό με τη σελίδα που θέλετε, στη γραμμή διευθύνσεων π.χ ένα όνομα, και πατήστε το &enterKey;. Αυτό θα σας οδηγήσει στο πρώτο αποτέλεσμα του Google για αυτή τη λέξη.

Πατώντας σε ένα δεσμό

Οι περισσότερες ιστοσελίδες περιέχουν δεσμούς στους οποίους μπορείτε να κάνετε κλικ για να μεταβείτε σε άλλες σελίδες.

  1. Μετακινήστε το δείκτη του ποντικιού μέχρι να αλλάξει και να γίνει δάκτυλο. Αυτό συμβαίνει κάθε φορά που ο δείκτης βρίσκεται πάνω από ένα δεσμό. Οι περισσότεροι δεσμοί έχουν υπογράμμιση, αλλά τα κουμπιά και οι εικόνες μπορεί να περιέχουν δεσμούς.
  2. Κάντε μια φορά κλικ στο δεσμό. Καθώς το δίκτυο θα εντοπίζει τη σελίδα του δεσμού, τα μηνύματα κατάστασης θα εμφανίζονται στο κάτω μέρος του παράθυρου

Δείτε το ιστορικό σας

Υπάρχουν αρκετοί τρόποι για να επισκεφτείτε ξανά μια σελίδα:

  • Για να μεταβείτε μπροστά ή πίσω μια σελίδα πατήστε τα κουμπιά Πίσω ή Μπροστά .
  • Για να μεταβείτε πίσω ή μπροστά σε περισσότερες από μια σελίδες, πατήστε στα μικρά τρίγωνα που βρίσκονται στα κουμπιά Πίσω και Μπροστά Εκεί θα δείτε μια λίστα των σελίδων που είδατε πρόσφατα. Για να επιστρέψετε σε μια σελίδα, επιλέξτε την από τη λίστα.
  • Για να δείτε μια λίστα των URLs που πληκτρολογήσατε στη γραμμή διευθύνσεων, κάντε κλικ στο βέλος στο δεξί άκρο της γραμμής διευθύνσεων. Για να δείτε μια σελίδα, επιλέξτε την από τη λίστα.
  • Για να επιλέξετε μια από τις σελίδες που έχετε επισκεφτεί στη τρέχουσα συνεδρία σας, ανοίξτε το μενού Ιστορικό και χρησιμοποιήστε τη λίστα στο κάτω μέρος του μενού.
  • Για να επιλέξετε μια από τις σελίδες που έχετε επισκεφτεί σε προηγούμενες συνεδρίες σας, ανοίξτε το μενού Ιστορικό και επιλέξτε Προβολή στη πλευρική στήλη Θα δείτε εκεί τη πλευρική στήλη του Ιστορικού, Το ιστορικό της πλευρικής στήλης εμφανίζει μια λίστα από φακέλους. Με κλικ στους φακέλους εμφανίζονται οι υποφάκελοι ή οι τίτλοι των ιστοσελίδων. Μπορείτε να κάνετε κλικ στο τίτλο μιας σελίδας για να δείτε αυτή τη σελίδα.

Διακοπή και ανανέωση

Αν μια σελίδα φορτώνει πολύ αργά ή δεν θέλετε πια να τη βλέπετε, κάντε κλικ στο κουμπί Διακοπή .

Για να ανανεώσετε τη τρέχουσα σελίδα ή για να δείτε τη πιο ενημερωμένη έκδοση της πατήστε στο κουμπί Ανανέωση ή πατήστε &accelKey;+R.

Περιήγηση με καρτέλες

Όταν επισκέπτεσθε περισσότερες από μια ιστοσελίδες τη φορά, μπορείτε να χρησιμοποιείτε τη Περιήγηση με καρτέλες για να κάνετε τα πάντα γρηγορότερα και ευκολότερα.

Η Περιήγηση με καρτέλες σας επιτρέπει να ανοίγετε καρτέλες που η κάθε μία θα εμφανίζει σελίδες μέσα σε ένα μόνο παράθυρο του &brandShortName;. Δεν θα χρειάζεται να έχετε πολλαπλά παράθυρα ανοικτά για να επισκέπτεσθε διαφορετικές σελίδες. Οι καρτέλες ελευθερώνουν χώρο στην επιφάνεια εργασίας σας. Μπορείτε να ανοίγετε, κλείνετε και να ανανεώνετε ιστοσελίδες εύκολα και σε ένα μέρος χωρίς να χρειάζεται να ψάχνετε σε διαφορετικά παράθυρα.

Για περισσότερες πληροφορίες, δείτε Περιήγηση με καρτέλες.

Χρησιμοποιώντας τη πλευρική στήλη

Η πλευρική στήλη είναι μια περιοχή στην αριστερή πλευρά της οθόνης, την οποία μπορείτε να την χρησιμοποιήσετε για τη προβολή των σελιδοδεικτών και του ιστορικού σας. Οι επεκτάσεις μπορούν να προσθέτουν νέους τρόπους για να χρησιμοποιήσετε τη πλευρική σας στήλη.

Για να δείτε ένα αντικείμενο στη πλευρική στήλη, επιλέξτε Προβολή > Πλευρική στήλη Από εκεί μπορείτε να επιλέξετε τη καρτέλα της πλευρικής στήλης που επιθυμείτε.

Αναζήτηση

Αναζήτηση στο διαδίκτυο

Η αναζήτηση για ιστοσελίδες από ένα συγκεκριμένο θέμα είναι πολύ εύκολη. Απλά πληκτρολογήστε μερικές λέξεις στη μπάρα αναζήτησης του &brandShortName;:

Για παράδειγμα αν θέλετε να βρείτε πληροφορίες για baby dolls:

  1. Κάντε κλικ στη μπάρα αναζήτησης
  2. Πληκτρολογήστε τη φράση baby doll. Το κείμενο που πληκτρολογείτε αντικαθιστά όποιο κείμενο υπάρχει στη μπάρα αναζήτηση.
  3. Πατήστε &enterKey; για αναζήτηση.

Τα αποτελέσματα αναζήτησης για "baby doll" θα εμφανίζονται στο παράθυρο &brandShortName;

Επιλογή της μηχανής αναζήτησης

Μπορείτε να αλλάξετε την μηχανή αναζήτησης κάνοντας κλικ στο εικονίδιο της και επιλέγοντας τη μηχανή αναζήτησης της επιλογής σας. Μερικές μηχανές αναζήτησης όπως το Google, κάνουν αναζήτηση σε όλο τον παγκόσμιο ιστό, ενώ άλλες όπως το Amazon.com, μόνο σε σε συγκεκριμένες ιστοσελίδες.

Διαχείριση μηχανών αναζήτησης.

Κάντε κλικ στο εικονίδιο της μηχανής αναζήτησης και επιλέξτε Διαχείριση μηχανών αναζήτησης... για να προσθέσετε, απομακρύνετε, ή να επαναφέρετε τις προεπιλεγμένες μηχανές αναζήτησης. Επιλέξτε μια μηχανή αναζήτησης και κάντε κλικ στο αντίστοιχο κουμπί για την μετακινήσετε ή να την διαγράψετε. Μπορείτε να εγκαταστήσετε νέες μηχανές αναζήτησης κάνοντας κλικ στο δεσμό Λήψη περισσότερων μηχανών αναζήτησης... .

Αναζήτηση για επιλεγμένες λέξεις σε μια ιστοσελίδα.

Ο &brandShortName; σας επιτρέπει να ψάξετε για λέξεις που επιλέγετε μέσα σε μια ιστοσελίδα:

  1. Επιλέξτε (με επισήμανση) τις λέξεις στην ιστοσελίδα.
  2. Κάντε δεξί κλικ Πατήστε το &ctrlKey;, και κλικ στο κουμπί του ποντικιού, και επιλέξτε και επιλέξτε [Μηχανή αναζήτησης] για "[οι επιλεγμένες σας λέξεις]" από το αναδυόμενο μενού.

Ο &brandShortName; ανοίγει μια νέα καρτέλα και χρησιμοποιεί τη προεπιλεγμένη μηχανή αναζήτησης για για να ψάξει για τις επιλεγμένες σας λέξεις.

Αναζήτηση μέσα στη σελίδα

Για να βρείτε ένα κείμενο μέσα στη σελίδα που βλέπετε στον &brandShortName;:

  1. και &accelKey;+F ή επιλέξτε Επεξεργασία > Εύρεση σε αυτή τη σελίδα... για να ανοίξετε την εργαλειοθήκη εύρεσης στο κάτω μέρος του &brandShortName;.
  2. Η αναζήτηση ξεκινάει αυτόματα μόλις πληκτρολογήσετε κάτι στο πλαίσιο αναζήτησης.
  3. Η εργαλειοθήκη εύρεσης σας προσφέρει τις ακόλουθες επιλογές:
    • Επόμενο: εύρεση κειμένου στη σελίδα το οποίο βρίσκεται μετά τη τρέχουσα θέση του δρομέα.
    • Προηγούμενο: εύρεση κειμένου στη σελίδα το οποίο βρίσκεται πριν τη τρέχουσα θέση του δρομέα.
    • Επισήμανση όλων: επισήμανση όλων των επαναλήψεων των όρων που ψάχνετε στη τρέχουσα σελίδα.
    • Ταίριασμα χαρακτήρα: περιορίστε την αναζήτηση στο κείμενο που έχει έχει μόνο τα ίδια κεφαλαία- μικρά με τον όρο που ψάχνετε.

Για να βρείτε ξανά την ίδια φράση ή λέξη: F3 ή επιλέξτε Επεξεργασία > Εύρεση ξανά.

Συμβουλή : Ενεργοποιήστε την &pref.singular; Αναζήτηση για κείμενο μόλις ξεκινήσει η πληκτρολόγηση στη καρτέλα Γενικά στην ενότητα Για προχωρημένους στις &pref.menuPath; για να ενεργοποιήσετε τη λειτουργία Γρήγορη εύρεση. Όταν ενεργοποιηθεί η Γρήγορη εύρεση ξεκινάει αυτοματα η αναζήτηση αυτών που πληκτρολογείτε. Σε αντίθεση με την εργαλειοθήκη εύρεσης η Γρήγορη εύρεση κλείνει αυτόματα μετά από λίγα δευτερόλεπτα αδράνειας.

Αντιγραφή, Αποθήκευση, και Εκτύπωση Ιστοσελίδων

Αντιγραφή ενός μέρους της σελίδας

Για να αντιγράψετε κείμενο από μια σελίδα:

  1. Επιλέξτε το κείμενο.
  2. Επιλέξτε Επεξεργασία > Αντιγραφή από το μενού.

Μπορείτε να κάνετε επικόλληση κειμένου σε άλλα προγράμματα.

Για να αντιγράψετε ένα δεσμό (URL) ή ένα δεσμό εικόνας από μια σελίδα:

  1. Τοποθετήστε τον δείκτη του ποντικιού σας πάνω στο δεσμό ή την εικόνα.
  2. Κάντε δεξί κλικ Πατήστε &ctrlKey; και κάντε κλικ στο το δεσμό ή την εικόνα για να εμφανιστεί ένα αναδυόμενο μενού.
  3. Επιλέξτε Αντιγραφή τοποθεσίας δεσμού, ή Αντιγραφή τοποθεσίας εικόνας. Αν μια εικόνα είναι και δεσμός, μπορείτε να επιλέξετε όποιο από τα δυο μενού θέλετε.

Μπορείτε να κάνετε επικόλληση σε άλλα προγράμματα ή στη γραμμή διευθύνσεων του &brandShortName;

Αποθήκευση ολόκληρης ή μέρους μιας σελίδας

Για να αποθηκεύσετε ολόκληρη τη σελίδα με τον &brandShortName;:

  1. Επιλέξτε Αρχείο > Αποθήκευση σελίδας ως. Θα δείτε να εμφανίζεται το πλαίσιο διαλόγου αποθήκευσης ως.
  2. Επιλέξτε μια τοποθεσία για την αποθηκευμένη σελίδα.
  3. Επιλέξτε ένα τύπο αρχείο για τη σελίδα που θέλετε να αποθηκεύσετε:
    • Ιστοσελίδα, Πλήρης : Αποθήκευση ολόκληρης της ιστοσελίδας μαζί με τις εικόνες. Αυτή η επιλογή σας επιτρέπει να βλέπετε χωρίς σύνδεση όλη τη σελίδα μαζί με τις εικόνες, αλλά δεν θα διατηρεί την δομή των δεσμών HTML της αρχικής σελίδας. Ο &brandShortName; δημιουργεί ένα νέο κατάλογο όπου αποθηκεύεται η σελίδα, οι εικόνες της και όλα τα άλλα απαραίτητα αρχεία για την εμφάνιση όλης της σελίδας.
    • Ιστοσελίδα, μόνο HTML : Αποθήκευση της αρχικής σελίδας χωρίς εικόνες. Αυτή η επιλογή διατηρεί την αρχική δομή HTML της σελίδας σε ένα αρχείο.
    • Αρχείο κειμένου: Αποθήκευση της σελίδας ως αρχείο κειμένου. Αυτή η επιλογή δεν διατηρεί τη δομή HTML, αλλά σας επιτρέπει να δείτε μια το κείμενο της σελίδας σε οποιονδήποτε επεξεργαστή κειμένου.
  4. Πληκτρολογήστε ένα όνομα αρχείου για τη σελίδα και κάντε κλικ Αποθήκευση .

Για να αποθηκεύσετε ένα πλαίσιο από μια ιστοσελίδα:

  1. Τοποθετήστε το δείκτη του ποντικιού σας μέσα στο πλαίσιο.
  2. Πατήστε το &ctrlKey; και κάντε κλικ στοΚάντε δεξί κλικ πλαίσιο για να εμφανιστεί ένα αναδυόμενο μενού.
  3. Επιλέξτε Αυτό το πλαίσιο > Αποθήκευση πλαισίου ως από το υπομενού. Θα δείτε το κουτί διαλόγου της Αποθήκευσης ως
  4. Επιλέξτε μια τοποθεσία για την αποθηκευμένη σελίδα.
  5. Επιλέξτε ένα τύπο για τη σελίδα που θέλετε να αποθηκεύσετε.
  6. Πληκτρολογήστε ένα όνομα αρχείου για τη σελίδα και κάντε κλικ Αποθήκευση .

Η αποθήκευση ενός αρχείου στο σκληρό σας δίσκο σας επιτρέπει να βλέπετε τη σελίδα ακόμα και αν δεν είστε συνδεδεμένοι στο διαδίκτυο.

Για να αποθηκεύσετε μια εικόνα από μια σελίδα:

  1. Τοποθετήστε το δείκτη του ποντικιού σας πάνω στην εικόνα.
  2. Κάντε δεξί κλικ Πατήστε &ctrlKey; και κάντε κλικ στο στην εικόνα για να εμφανιστεί ένα αναδυόμενο μενού.
  3. Επιλέξτε Αποθήκευση εικόνας ως Θα δείτε το πλαίσιο διαλόγου της αποθήκευσης εικόνας.
  4. Επιλέξτε μια τοποθεσία για την αποθηκευμένη εικόνα.
  5. Πληκτρολογήστε ένα όνομα αρχείου για την εικόνα και κάντε κλικ Αποθήκευση .

Για μα αποθηκεύσετε ένα αρχείο ή σελίδα χωρίς να το εμφανίσετε (χρήσιμο για την λήψη μιας μη μορφοποιημένης σελίδας ή ενός αρχείου που δεν έχει σχεδιαστεί για εμφάνιση):

  1. Τοποθετήστε το δείκτη του ποντικιού σας πάνω σε ένα δεσμό στη σελίδα.
  2. Κάντε δεξί κλικ Πατήστε &ctrlKey; και κάντε κλικ στο στο δεσμό για να εμφανιστεί ένα αναδυόμενο μενού.
  3. Επιλέξτε Αποθήκευση δεσμού ως Θα δείτε το πλαίσιο διαλόγου της αποθήκευσης ως.
  4. Επιλέξτε μια τοποθεσία για την αποθηκευμένη σελίδα.
  5. Πληκτρολογήστε ένα όνομα αρχείου για τη σελίδα και κάντε κλικ Αποθήκευση .

Σημαντικό: Μερικοί δεσμοί οδηγούν σε αυτόματη λήψη και αποθήκευση αρχείων στο σκληρό σας δίσκο, όταν κάνετε κλικ σε αυτούς. Αυτά τα URLs συχνά ξεκινάνε με "ftp" ή τελειώνουν με επέκταση όπως "au" ή "mpeg.". Αυτοί οι δεσμοί μπορεί να περιέχουν λογισμικό, ήχους, ταινίες και πιθανόν να εκκινούν βοηθητικές εφαρμογές που υποστηρίζουν αυτά τα αρχεία.

Συμβουλή : Για να βάλετε μια εικόνα ως ταπετσαρία της επιφάνειας εργασίας σας, κάντε δεξί κλικ πατήστε το πλήκτρο &ctrlKey;, κλικ στο κουμπί του ποντικιού σε μια εικόνα και επιλέξτε Ορισμός ως ταπετσαρία... από το αναδυόμενο μενού.

Εκτύπωση μιας σελίδας

Για να εκτυπώσετε τη τρέχουσα σελίδα:

  • Επιλέξτε Αρχείο > Εκτύπωση.

Για να εκτυπώσετε κείμενο που έχετε επιλέξει:

  • Επιλέξτε το κείμενο στη σελίδα.
  • Επιλέξτε Αρχείο > Εκτύπωση. Ο διάλογος εκτύπωσης θα εμφανιστεί.
  • Κάτω Σειρά εκτύπωσης , κάντε κλικ Επιλογή .

Η τοποθέτηση του περιεχομένου στην προς εκτύπωση σελίδα, καθορίζεται από τον συντάκτη και το μέγεθος της ιστοσελίδας και όχι από το μέγεθος τους παραθύρου που είναι ορατό στην οθόνη σας. Το κείμενοέχει αναδιπλωθεί και τα γραφικά έχουν τοποθετηθεί διαφορετικά ώστε να χωράνε στο μέγεθος του χαρτιού.

Χρήση της προεπισκόπησης εκτύπωσης

Για να έχετε μια ιδέα για το πως θα δείχνει η σελίδα πριν εκτυπωθεί, μπορείτε να χρησιμοποιήσετε την Προεπισκόπηση εκτύπωσης:

  • Επιλέξτε Αρχείο > Προεπισκόπηση εκτύπωσης.

Στη Προεπισκόπηση εκτύπωσης, μπορείτε να κάνετε τα παρακάτω με τις σελίδες που θέλετε να εκτυπώσετε:

  • Προβολή μιας προεπισκόπησης της κάθε σελίδας : Πατήστε το κουμπί Επόμενο , Προηγούμενο , Πρώτη σελίδα , ή Τελευταία σελίδα για να μετακινηθείτε ανάμεσα στις σελίδες.
  • Αλλαγή της κλίμακας (μεγέθους) των σελίδων : Στο μενού Κλίμακα επιλέξτε "50%" για να μειώσετε στο μισό το μέγεθος της αρχικής σελίδας. Για να αλλάζετε αυτόματα το μέγεθος των σελίδων για να χωράνε στο πλάτος του χαρτιού επιλέξτε "Σμίκρυνση στο πλάτος της σελίδας". Μπορείτε ακόμα να ορίσετε το δικό σας ποσοστό επιλέγοντας "Προσαρμοσμένο...".
  • Αλλαγή του προσανατολισμού της σελίδας: Επιλέξτε Κατακόρυφος για να τοποθετήσετε τη σελίδα ώστε η μικρότερη πλευρά της να δείχνει προς τα πάνω. Επιλέξτε Οριζόντιος για να τοποθετήσετε τη σελίδα ώστε η μεγαλύτερη της πλευρά να δείχνει προς τα πάνω.
  • Μετάβαση στη Διαμόρφωση σελίδας: Πατήστε Διαμόρφωση σελίδας για τη περαιτέρω παραμετροποίηση των σελίδων που θέλετε να εκτυπώσετε.
  • Μετάβαση στην Εκτύπωση: Πατήστε Εκτύπωση για να εκτυπώσετε τις σελίδες.

Σημείωση: Μερικές λειτουργίες της Προεπισκόπησης εκτύπωσης είναι διαφορετικές ή μη διαθέσιμες στο Mac OS και στο Linux.

Χρήση της Διαμόρφωσης σελίδας

Σημείωση: Μερικές λειτουργίες της Διαμόρφωσης σελίδας είναι διαφορετικές ή μη διαθέσιμες στο Mac OS και στο Linux.

Για να παραμετροποιήσετε το τρόπο εκτύπωσης των σελίδων στον &brandShortName;, μπορείτε να χρησιμοποιήσετε τη Διαμόρφωση σελίδας:

  • Επιλέξτε Αρχείο > Διαμόρφωση σελίδας.

Στη Διαμόρφωση σελίδας, μπορείτε να αλλάξετε τις ακόλουθες ρυθμίσεις για τις σελίδες που θέλετε να εκτυπώσετε:

  • Μορφή & Επιλογές: Επιλογή του προσανατολισμού, κλίμακας και άλλων επιλογών:
    • Προσανατολισμός:
      • Κατακόρυφος: Επιλέξτε αυτό για να τοποθετήσετε τη σελίδα κανονικά, με τη μικρότερη πλευρά προς τα πάνω.
      • Οριζόντιος : Επιλέξτε αυτό για να τοποθετήσετε τη σελίδα πλάγια, με τη μεγαλύτερη πλευρά προς τα πάνω.
    • Κλίμακα : Πληκτρολογήστε ένα ποσοστό της αρχικής σελίδας. Για παράδειγμα, πληκτρολογήστε "50" για να μικρύνετε κάθε σελίδα στο μισό της αρχικής σελίδας.
      • Σμίκρυνση στο πλάτος της σελίδας: Επιλέξτε αυτό ώστε να αλλάζει αυτόματα το μέγεθος της σελίδας για να ταιριάζει στο πλάτος του χαρτιού
    • Επιλογές :
      • Εκτύπωση παρασκηνίου (χρώματα και εικόνες) : Επιλέξτε αυτό για να εκτυπώνετε εικόνες παρασκηνίου και χρώματα. Αν δεν επιλεχθεί θα εκτυπώνονται μόνο εικόνες και χρώματα που βρίσκονται στο προσκήνιο.
  • Περιθώρια & Κεφαλίδα/Υποσέλιδο: Πατήστε σε αυτή την καρτέλα για να ρυθμίσετε περιθώρια, κεφαλίδες και υποσέλιδα:
    • Περιθώρια:
      • Πάνω, Κάτω, Δεξιά, Αριστερά : Πληκτρολογήστε ένα μήκος περιθωρίου για το πάνω, κάτω, δεξιό και αριστερό περιθώριο.
    • Κεφαλίδες & Υποσέλιδα: Κάθε κυλιόμενη λίστα αντιπροσωπεύει μια περιοχή κεφαλίδας ή υποσέλιδου. Η πάνω γραμμή των κυλιόμενων λιστών είναι για τις περιοχές της αριστερής, κεντρικής και δεξιάς κεφαλίδας. Η κάτω γραμμή είναι για τις περιοχές τού αριστερού, κεντρικού και δεξιού υποσέλιδου.Σε κάθε κυλιόμενη λίστα επιλέξτε μια από τις παρακάτω επιλογές:
      • --κενή--: Δεν εμφανίζεται τίποτα σε αυτή τη περιοχή.
      • Τίτλος: Εμφανίζεται ο τίτλος της ιστοσελίδας.
      • URL: Εμφανίζεται το URL της ιστοσελίδας (τα URL συνήθως ξεκινάνε με "http://").
      • Ημερομηνία/Ώρα: Εμφανίζεται η ημερομηνία και η ώρα εκτύπωσης της ιστοσελίδας.
      • Σελίδα #: Εμφανίζεται ο αριθμός σελίδας για κάθε σελίδα.
      • Σελίδα # από #: Εμφανίσει τον αριθμό της σελίδας μαζί με το συνολικό αριθμό των σελίδων του εγγράφου. Για παράδειγμα αν εκτυπώσετε μια ιστοσελίδα πέντε σελίδων, το "3 από 5" θα εμφανίζεται στη τρίτη σελίδα
      • Προσαρμοσμένη: Πληκτρολογήστε το δικό σας κείμενο. Μπορείτε να συμπεριλάβετε οποιονδήποτε από τους παρακάτω κωδικούς για να εκτυπώσετε συγκεκριμένες πληροφορίες:
        • &PT: Αριθμός σελίδας με σύνολο σελίδων (Παράδειγμα: "3 από 5")
        • &P: Αριθμός σελίδας
        • &D: Ημερομηνία
        • &U: URL
        • &T: Τίτλος σελίδας

Συμβουλή : Για να δείτε μια προεπισκόπηση των αλλαγών που κάνατε μέσω της Διαμόρφωσης σελίδας, χρησιμοποιήστε Προεπισκόπηση εκτύπωσης.

Βελτιώνοντας την ταχύτητα και την αποτελεσματικότητα

Διαχείριση διαφορετικών τύπων αρχείων

Ο &brandShortName; μπορεί να χειρίζεται πολλούς διαφορετικούς τύπους αρχείων. Όμως για μερικά αρχεία όπως ταινίες ή μουσική, ο &brandShortName; χρειάζεται Πρόσθετες λειτουργίες ή εξωτερικές εφαρμογές που μπορούν να χειριστούν αυτά τα αρχεία. Αν ο &brandShortName; δεν έχει την απαιτούμενη πρόσθετη λειτουργία ή δεν γνωρίζει ποια εξωτερική εφαρμογή ανοίγει το αρχείο, μπορείτε να αποθηκεύσετε το αρχείο στο δίσκο σας. Μπορείτε να έχετε εικόνα της αποθήκευσης των αρχείων,μέσω της Διαχείρισης λήψης αρχείων.

Οι πρόσθετες λειτουργίες προσθέτουν στον &brandShortName; επιπρόσθετη λειτουργικότητα και λειτουργούν εσωτερικά στην εφαρμογή. Πρόσθετες λειτουργίες όπως οι Sun Java, Macromedia Flash, και RealNetworks RealPlayer επιτρέπουν στον &brandShortName; να εμφανίζει αρχεία πολυμέσων και να εκτελεί μικρές εφαρμογές όπως ταινίες κινούμενα σχέδια και παιχνίδια. Μπορείτε να βρείτε τις πιο κοινές πρόσθετες λειτουργίες στη Σελίδα πρόσθετων λειτουργιών στο Mozilla Update.

Όταν τα αρχεία δε μπορούν να ανοιχθούν με τον &brandShortName;, μπορείτε να επιλέξετε να ανοίγουν με εφαρμογές έξω από τον &brandShortName;. Για παράδειγμα μπορείτε να χρησιμοποιήσετε την εφαρμογή αναπραγωγής πολυμέσων της επιλογής σας για αρχεία MP3.

Μπορείτε να καθορίσετε τον τρόπο που ο &brandShortName; θα χειρίζεται τα αρχεία που δεν μπορεί να εμφανίσει στο Ενέργειες λήψης αρχείων διάλογο. Για να ανοίξετε αυτό το διάλογο, επιλέξτε &pref.menuPath;, ανοίξτε το τμήμα Περιεχόμενο και κάντε κλικ στο Διαχείριση που βρίσκεται στο Τύποι αρχείων Μπορείτε να αλλάξετε την ενέργεια για ένα τύπο αρχείου ή να την απομακρύνετε. Περισσότερες πληροφορίες για αυτό τον διάλογο είναι διαθέσιμες στην αντίστοιχη ενότητα του &pref.plural; Τύποι αρχείων.

Όταν κάνετε κλικ σε ένα αρχείο και ο &brandShortName; δεν ξέρει πως να το χειριστεί, ανοίγει ένα παράθυρο διαλόγου, που παρουσιάζει τις εξής επιλογές:

  • Άνοιγμα με: Ο &brandShortName; ανοίγει το αρχείο χρησιμοποιώντας το προεπιλεγμένο πρόγραμμα από τις ρυθμίσεις του συστήματος σας. Μπορείτε ακόμα να επιλέξετε ένα διαφορετικό πρόγραμμα επιλέγοντας Άλλο... από τη κυλιόμενη λίστα.
  • Αποθήκευση στο δίσκο: Ο &brandShortName; σας επιτρέπει να επιλέξετε μια τοποθεσία για να σποθηκεύσετε το αρχείο. Αν έχετε επιλέξει Αποθήκευση όλων των αρχείων σε αυτόν το φάκελο από το Βασικές πίνακα &pref.pluralCaps;, το αρχείο θα αποθηκευθεί αυτόματα στο προεπιλεγμένο φάκελο. Αλλιώς ο &brandShortName; θα σας ρωτήσει για την τοποθεσία αποθήκευσης.
  • Να γίνεται αυτό αυτόματα για αρχεία αυτού του είδους : Όταν αυτό επιλεχθεί, ο &brandShortName; θα εκτελεί αυτόματα την επιλεγμένη ενέργεια την επόμενη φορά που θα κάνετε λήψη ένα αρχείο αυτού του τύπου.

Συμβουλή : Μπορείτε να διαγράψετε τις αποθηκευμένες σας &pref.plural; για Να γίνεται αυτό αυτόματα για αρχείο αυτού του είδους από το Ενέργειες Λήψης αρχείων διάλογος.

Ορισμός του &brandShortName; ως τον προεπιλεγμένο σας περιηγητή

Ο &brandShortName; προβάλλει ιστοσελίδες από το διαδίκτυο και από τον υπολογιστή σας. Για να ανοίγετε εύκολα ιστοσελίδες, μπορείτε να κάνετε τον &brandShortName; τον προεπιλεγμένο σας περιηγητή.

Για να κάνετε τον &brandShortName; τον προεπιλεγμένο σας περιηγητή, επιλέξτε &pref.menuPath; και κάντε κλικ στο κουμπί 'Έλεγχος τώρα που βρίσκεται στο Βασικές Γενικά.

Αλλαγή ρυθμίσεων λανθάνουσας μνήμης

Ο &brandShortName; αποθηκεύει αντίγραφα από τις σελίδες που επισκέπτεστε συχνά στη λανθάνουσα μνήμη. Έτσι ο &brandShortName; δεν χρειάζεται να κάνει λήψη της σελίδας από το δίκτυο κάθε φορά που βλέπετε τη σελίδα.

Για να ορίσετε το μέγεθος της λανθάνουσας μνήμης:

  1. Επιλέξτε &pref.menuPath;.
  2. Ανοίξτε την ενότητα Για προχωρημένους και κάντε κλικ στη καρτέλα Δίκτυο .
  3. Εισάγετε έναν αριθμό στο πεδίο Χρήση μέχρι πεδίο για να καθορίσετε το μέγεθος της λανθάνουσας μνήμης, Η προεπιλογή είναι 50MB.

Σημαντικό: : Μια μεγαλύτερη σε μέγεθος λανθάνουσα μνήμη επιτρέπει την πιο γρήγορη ανάκτηση σελίδων, αλλά χρησιμοποιεί και περισσότερο χώρο στο δίσκο.

Όταν κλείνετε τον &brandShortName;, γίνεται μια τακτοποίηση της λανθάνουσας μνήμης. Αν η τακτοποίηση διαρκεί περισσότερο χρόνο από όσο εσείς επιθυμείτε, μειώστε το μέγεθοςτης λανθάνουσας μνήμης

Για να ανανεώσετε οποιαδήποτε στιγμή μια σελίδα, πατήστε το κουμπί Ανανέωση στη εργαλειοθήκη πλοήγησης του &brandShortName;. Τότε ο &brandShortName; φορτώνει και εμφανίζει τη πιο πρόσφατη έκδοση της σελίδας.
