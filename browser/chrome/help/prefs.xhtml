Βασικές &pref.pluralCaps;

Εκκίνηση

Όταν ανοίγει ο &brandShortName; στην αρχή η Αρχική σελίδα εμφανίζεται. Η αρχική σας σελίδα μπορεί να είναι μια σελίδα που επισκέπτεσθε συχνά, μια μηχανή αναζήτησης ή ακόμα και ο λογαριασμός του email σας, αλλά η επιλογή είναι δική σας. Μπορείτε να αλλάξετε την αρχική σας σελίδα ή να επιλέξετε να εμφανίζεται μια διαφορετική σελίδα ή ομάδα από σελίδες στην εκκίνηση,

Στην εκκίνηση του &brandShortName;
Όταν ο &brandShortName; ξεκινάει ανοίγει την αρχική σας σελίδα (που είναι στον παρακάτω πλαίσιο διαλόγου), που αντιστοιχεί στην επιλογή Προβολή της αρχικής μου σελίδας. Εναλλακτικά μπορείτε να επιλέξτε να εμφανίζεται μια κενή σελίδα στην εκκίνηση (για να μειώσετε το χρόνο φόρτωσης της σελίδας από το διαδίκτυο) επιλέγοντας την επιλογή Προβολή μιας κενής σελίδας .

Τέλος έχετε την δυνατότητα να ανοίγει ο &brandShortName; ξανά τα παράθυρα και τις καρτέλες που είχατε στην τελευταία σας συνεδρία με τον &brandShortName;, και να σας επαναφέρει στην τελευταία σας κατάσταση όταν διακόψατε την περιήγηση σας. Αυτός είναι ένας εύχρηστος τρόπος αποθήκευσης της τελευταίας σας συνεδρίας πριν να κλείσετε τον &brandShortName;. Μπορείτε να επιλέξετε αυτή την προτίμηση κάνοντας κλικ στο Προβολή των παραθύρων και των καρτελών που υπήρχαν την τελευταία φορά.

Αρχική σελίδα
Αν έχετε καθορίσει να δείχνει ο &brandShortName; την αρχική σας σελίδα στην εκκίνηση του &brandShortName;, μπορείτε να ορίσετε την αρχική σας σελίδα πληκτρολογώντας την εδώ. Μπορείτε ακόμα να κάνετε κλικ σε κάποιο από τα παρακάτω κουμπιά για να επιλέξτε μια ειδική ιστοσελίδα.

  • Χρήση Τρέχουσας Σελίδας ορίζει την τρέχουσα σελίδα (ή σελίδες, αν έχετε πολλές καρτέλες ανοικτές) ως την αρχική σας σελίδα
  • Χρήση σελιδοδείκτη ορίζει τον επιλεγμένο σελιδοδείκτη (ή αν επιλέξετε ένα φάκελο, μια ομάδα σελιδοδεικτών) ως την αρχική σας σελίδα
  • Επαναφορά στη προεπιλογή επαναφέρει την αρχική σας σελίδα στην προεπιλεγμένη αρχική σελίδα

Λήψεις αρχείων

Αυτή η ενότητα περιέχει τις &pref.plural; που καθορίζουν τον τρόπο λήψης αρχείων με τον &brandShortName;.

Εμφάνιση της Διαχείρισης λήψης αρχείων όταν ξεκινά μια λήψη
Με αυτή την &pref.singular; επιλεγμένη, ο &brandShortName; θα ανοίγει το Παράθυρο λήψης αρχείων όταν ξεκινάει η λήψη ενός αρχείου.

Κλείσιμο του όταν ολοκληρωθούν όλες οι λήψεις
Επιλέξτε αυτή την επιλογή ώστε ο &brandShortName; να κλείνει το παράθυρο λήψεων όταν ολοκληρώνεται όλες ο λήψεις αρχείων.

Αποθήκευση αρχείων σε
Αυτή είναι η προεπιλεγμένη προτίμηση του Firefox, και ο προεπιλεγμένος φάκελος στην Επιφάνεια εργασίας σας. Μπορείτε να καθορίσετε ένα διαφορετικό φάκελο όπου θα αποθηκεύονται όλες οι λήψεις, κάνοντας κλικ στο Εξερεύνηση .

Πάντα ερώτηση για την τοποθεσία αποθήκευσης κάθε αρχείου
Όταν αυτή η &pref.singular; επιλεχθεί, ο &brandShortName; θα σας κάνει ερώτηση για το που θέλετε να αποθηκεύσετε κάθε αρχείο που λαμβάνετε (αντί να χρησιμοποιεί την προεπιλεγμένη τοποθεσία που έχει καθορισθεί παραπάνω)

Προεπιλογές συστήματος

Ο &brandShortName; θα ελέγχει κατά την εκκίνηση αν είναι ο προεπιλεγμένος περιηγητής.
Επιλέξτε αυτή την επιλογή, αν θέλετε να ελέγχει ο &brandShortName; να ελέγχει αν είναι ο προεπιλεγμένος περιηγητής στην εκκίνηση. Αυτό θα διασφαλίσει την χρήση του &brandShortName; κάθε φορά που μια εφαρμογή προσπαθεί να συνδεθεί σε μια ιστοσελίδα, ή όταν θέλετε να ανοίξετε ένα αρχείο HTML. Μπορείτε ακόμα να κάνετε κλικ στο κουμπί Έλεγχος τώρα για να κάνετε έλεγχο άμεσα.

&pref.pluralCaps; Καρτελών

Οι νέες σελίδες να ανοίγουν σε:
Αυτή η &pref.singular; ελέγχει για το αν οι δεσμοί από άλλες εφαρμογές ή από ιστοσελίδες που ζητούν να ανοίξουν νέα παράθυρα, θα ανοίγουν σε νέα παράθυρα , ή μια νέα καρτέλα στο πιο πρόσφατο παράθυρο

Σημείωση: Αν έχετε επιλέξει να ανοίγουν οι δεσμοί σε νέες καρτέλες, ο &brandShortName; θα αγνοεί αυτή την &pref.singular; και θα ανοίγει ένα νέο παράθυρο, όταν ο δημιουργός της σελίδας έχει καθορίσει ότι το νέο παράθυρο θα ανοίγει σε συγκεκριμένο μέγεθος, επειδή μερικές ιστοσελίδες μπορούν αν εμφανιστούν σωστά μόνο σε ένα συγκεκριμένο μέγεθος.

Προειδοποίηση στο κλείσιμο πολλαπλών καρτελών
Όταν κλείνετε ένα παράθυρο με πολλαπλές καρτέλες, ο &brandShortName; θα ζητάει την επιβεβαίωση σας. Αυτό εμποδίζει το κατά λάθος κλείσιμο ενός παραθύρου με χρήσιμες καρτέλες. Αποεπιλέξτε αυτή την επιλογή για να παρακάμψετε την εμφάνιση των προειδοποιήσεων, και να κάνετε τον &brandShortName; να κλείνει αυτόματα το παράθυρο.

Προειδοποίηση στο κλείσιμο πολλαπλών καρτελών που μπορεί να επιβραδύνουν τον &brandShortName;
Όταν κλείνετε ένα παράθυρο με πολλαπλές καρτέλες, ο &brandShortName; θα ζητάει την επιβεβαίωση σας. Αυτό εμποδίζει την επιβράδυνση της λειτουργίες του συστήματος σας όταν ανοίγουν πολλές καρτέλες. Αποεπιλέξτε αυτή την &pref.singular; για να παρακάμψετε την εμφάνιση των προειδοποιήσεων.

Να εμφανίζεται πάντα η μπάρα καρτελών
Αν βλέπετε μόνο μια ιστοσελίδα σε ένα παράθυρο του &brandShortName;, η μπάρα καρτελών δεν θα εμφανίζεται. Επιλέξτε αυτή την &pref.singular; για να εμφανίζεται πάντα η μπάρα καρτελών ακόμα και αν το παράθυρο του &brandShortName; περιέχει μόνο μα σελίδα.

Άμεση εστίαση στο άνοιγμα ενός νέου δεσμού σε μια νέα καρτέλα
Όταν κάνετε μεσαίο κλικ σε δεσμούς (ή πατάτε το &accelKey; ενώ κάνετε κλικ με το αριστερό κουμπί του ποντικιού), οι δεσμοί θα ανοίγουν σε νέα καρτέλα. Αυτή η καρτέλα δεν θα εμφανίζεται άμεσα, αλλά θα φορτώνεται στο παρασκήνιο. Επιλέξτε αυτή τη &pref.singular;, για να φορτώνεται ο δεσμός σε καρτέλα στο προσκήνιο και να εμφανίζεται άμεσα η καρτέλα.

&pref.pluralCaps; Περιεχομένου

Φραγή αναδυομένων παραθύρων
Ο &brandShortName; εξ ορισμού κάνει φραγή των ενοχλητικών αναδυόμενων παραθύρων σε ιστοσελίδες. Αν αποεπιλέξετε αυτή τη &pref.singular; θα απενεργοποιηθεί και η φραγή αναδυομένων παραθύρων.

Μερικές ιστοσελίδες κάνουν νόμιμη χρήση των αναδυόμενων παραθύρων. Για να επιτρέψετε σε αυτές τις σελίδες να ανοίγουν αναδυόμενα παράθυρα κάντε κλικ στο Εξαιρέσεις..., εισάγετε το όνομα της σελίδας και κάντε κλικ στο Να επιτρέπεται Για να απομακρύνετε μια σελίδα από τη λίστα, επιλέξτε την και κάντε κλικ στο Απομάκρυνση σελίδας. Για να διαγράψετε οριστικά τη λίστα, κάντε κλικ στο Απομάκρυνση όλων των σελίδων .

Αυτόματη φόρτωση εικόνων.
Οι εικόνες φορτώνονται πάντα στον &brandShortName; με τις ιστοσελίδες. Αποεπιλέξτε αυτή την &pref.singular; για να απενεργοποιήσετε ολοκληρωτικά τις εικόνες.

Αν ενεργοποιήσετε την αυτόματη φόρτωση εικόνων, το Εξαιρέσεις... σας επιτρέπει να επιλέξετε από ποιες σελίδας δεν θα επιτρέπεται η αυτόματη φόρτωση εικόνων. Το κουμπί Εξαιρέσεις... σας επιτρέπει να καθορίσετε ποιες ιστοσελίδες επιθυμείτε να φορτώνουν (ή να μην φορτώνουν) ανεξάρτητα από τη ρύθμιση αυτής της προτίμησης. Στη λίστα εξαιρέσεων, εισάγετε τη σελίδα που θέλετε να επιτρέψετε να επιτρέπει η να κάνει φραγή εικόνων και κάντε κλικ στο κουμπί Να επιτρέπεται για να επιτρέπονται εικόνες, ή κάντε κλικ στο Φραγή για να κάνετε φραγή των εικόνων.

Γραμματοσειρές & Χρώματα

Προεπιλεγμένη γραμματοσειρά και Μέγεθος
Οι ιστοσελίδες συνήθως προβάλλονται με τη γραμματοσειρά και το μέγεθος που καθορίζεται εδώ. Όμως, μερικές ιστοσελίδες μπορούν να παρακάμψουν αυτές τις επιλογές εκτός και αν έχετε κάνει διαφορετική επιλογή στο διάλογο Για προχωρημένους στις γραμματοσειρές. Κάντε κλικ στο κουμπί Για προχωρημένους για να έχετε πρόσβαση σε περισσότερες &pref.plural; γραμματοσειρών

Διάλογος γραμματοσειρών

  1. Από τη κυλιόμενη λίστα Γραμματοσειρές για, και επιλέξτε μια ομάδα γλώσσας/γραφής. Για παράδειγμα, για να ορίσετε τις προεπιλεγμένες γραμματοσειρές για την Ελληνική γλώσσα/γραφή, επιλέξτε Ελληνική . Για μια γλώσσα/γραφή που δεν είναι ακόμα στη λίστα, επιλέξτε Άλλες γλώσσες .
  2. Επιλέξτε αν το μεταβλητού μήκους κείμενο θα είναι serif (όπως "Times New Roman") ή sans-serif (όπως "Arial"). Μετά καθορίστε το μέγεθος γραμματοσειράς για το σταθερού μήκους κείμενο.
  3. Καθορίστε τη γραμματοσειρά που θα χρησιμοποιείται για τις γραμματοσειρές Serif, Sans-serif και Monospace. Μπορείτε ακόμα να αλλάξετε το μέγεθος για Γραμματοσειρές Monospace.

Ακόμα, μπορείτε να καθορίσετε το ελάχιστο μέγεθος γραμματοσειράς που θα χρησιμοποιείται στην οθόνη. Αυτό μπορεί να φανεί χρήσιμο όταν μερικές ιστοσελίδες χρησιμοποιούν πολύ μικρές γραμματοσειρές και είναι δυσανάγνωστες.

Οι ιστοσελίδες να χρησιμοποιούν τις δικές τους γραμματοσειρές
Ο &brandShortName; χρησιμοποιεί τις γραμματοσειρές που καθορίζονται από την ιστοσελίδα. Αν απενεργοποιήσετε αυτή τη &pref.singular;, θα εξαναγκάσετε όλες τις σελίδες να εμφανίζονται με τις γραμματοσειρές που έχετε επιλέξει ως προεπιλεγμένες.

Κωδικοποίηση χαρακτήρων
Η κωδικοποίηση χαρακτήρων που επιλέγετε εδώ θα χρησιμοποιηθεί για την προβολή ιστοσελίδων που δεν καθορίζουν κωδικοποίηση χαρακτήρων.

Διάλογος χρωμάτων

Κείμενο και παρασκήνιο
Εδώ μπορείτε να αλλάξετε το προεπιλεγμένο χρώμα του κειμένου και του παρασκηνίου που θα χρησιμοποιείται σε σελίδες που δεν παρέχουν τέτοιες πληροφορίες, Κάντε κλικ στο χρωματολόγιο για να επιλέξετε χρώμα.

Χρήση χρωμάτων συστήματος
Επιλέξτε αυτή την &pref.singular; για να χρησιμοποιήσετε τα προεπιλεγμένα χρώματα του λειτουργικού σας αντί για τα χρώματα που καθορίζονται παραπάνω.

Χρώματα δεσμών
Εδώ μπορείτε να αλλάξετε τα προεπιλεγμένα χρώματα για τους δεσμούς. Κάντε κλικ στο χρωματολόγιο για να επιλέξετε χρώματα.

Υπογράμμιση δεσμών
Οι δεσμοί είναι υπογραμμισμένοι στις ιστοσελίδες. Αποεπιλέξτε αυτή την &pref.singular; για να απενεργοποιήσετε την υπογράμμιση, Σημειώσετε ότι πολλές ιστοσελίδες καθορίζουν τις δικές τους μορφοποιήσεις και αυτή η &pref.singular; δεν έχει κανένα αποτέλεσμα σε αυτές τις σελίδες.

Να επιτρέπεται η χρήση χρωμάτων στις σελίδες, αντί των επιλογών μου
Ο &brandShortName; χρησιμοποιεί τα χρώματα που καθορίζονται από την ιστοσελίδα. Αν απενεργοποιήσετε αυτή την &pref.singular; θα εξαναγκάσετε τις σελίδες να προβάλλονται με τα προεπιλεγμένα σας χρώματα.

Τύποι αρχείων

Ο διάλογος ενεργειών λήψης αρχείων, μπορεί να ανοιχθεί κάνοντας κλικ στο κουμπί Διαχείριση... που περιέχει διάφορους τύπους αρχείων. Μπορείτε να επιλέξετε για το τι θα κάνει ο &brandShortName; όταν κάνετε κλικ σε κάποιο τύπο αρχείων, επιλέγοντας το τύπο αρχείου που θέλετε να τροποποιήσετε και κάντε κλικ στο Αλλαγή ενέργειας... .

Αυτό θα εμφανίσει το διάλογο Αλλαγή ενέργειας, όπου και μπορείτε να επιλέξετε να ανοίγει ένας τύπος αρχείου με την προεπιλεγμένη εφαρμογή, με μια εφαρμογή επιλεγμένη από το χρήστη, να την αποθηκεύετε στο δίσκο ή να ανοίγει με μια εγκατεστημένη πρόσθετη λειτουργία. Για παράδειγμα, αν βλέπετε πολλά αρχεία πολυμέσων μέσα από ιστοσελίδες, μπορείτε να ρυθμίσετε τον &brandShortName; να ανοίγει πάντα αυτά τα αρχεία με την εφαρμογή αναπαραγωγής πολυμέσων σας, αντί να αποθηκεύει κάθε αρχείο.

  • Άνοιγμα τους με την προεπιλεγμένη εφαρμογή:
    Επιλέξτε αυτή την &pref.singular;, για να ανοίγει αυτός ο τύπος αρχείων με την προεπιλεγμένη εφαρμογή (που καθορίζεται από το λειτουργικό σας σύστημα).
  • Άνοιγμα τους με αυτή την εφαρμογή:
    Επιλέξτε αυτή την &pref.singular; για να ορίσετε μια άλλη εφαρμογή που θα χειρίζεται αυτό το τύπο αρχείου. Θα δείτε ένα διάλογο που σας ζητάει να καθορίσετε ποια εφαρμογή θα χρησιμοποιηθεί.
  • Αποθήκευση τους στον υπολογιστή μου:
    Με αυτή τη &pref.singular; όλα τα αρχεία θα αποθηκεύονται στο δίσκο. Αν έχετε την &pref.singular; Αποθήκευση αρχείων σε &pref.singular; στην ενότητα Βασικές
  • Χρήση αυτής της πρόσθετης λειτουργίας:
    Επιλέξτε αυτή την &pref.singular; για να επιτρέψετε σε μια πρόσθετη λειτουργία να χειρίζεται αυτό τον τύπο αρχείου.

&pref.pluralCaps; Αρχείων ροής

Τα αρχεία ροής είναι ειδικά διαμορφωμένα έγγραφα περίληψης του περιεχομένου μιας ιστοσελίδας. Για παράδειγμα, ένα αρχείο ροής μπορεί να κάνει περίληψη των τελευταίων ειδήσεων από μια ειδησεογραφική ιστοσελίδα ή τις τελευταίες δημοσιεύσεις από ένα ιστολόγιο. Μπορείτε να κάνετε προεπισκόπηση των περιεχομένων των αρχείων ροής στον υπολογιστή σας ή στο διαδίκτυο. Αυτή η &pref.plural; καθορίζει τον χειρισμό των αρχείων ροής από τον &brandShortName;.

Προβολή προεπισκόπησης και ερώτηση για το ποια εφαρμογή θα χρησιμοποιηθεί
Όταν βλέπετε ένα αρχείο ροής με τον &brandShortName;, θα υπάρχει πάντα μια προεπισκόπηση των περιεχομένων του. Με αυτή την &pref.singular; επιλεγμένη, θα σας δίνεται πάντα η δυνατότητα στο πάνω μέρος της σελίδας προεπισκόπησης, για τον αν θέλετε να κάνετε εγγραφή σε αυτό το αρχείο ροής.

Συνδρομή στο αρχείο ροής με χρήση

Όταν αυτή η &pref.singular; επιλεχθεί, οι ενημερώσεις του &brandShortName; θα λαμβάνονται αυτόματα και θα εγκαθίστανται στην επόμενη επανεκκίνηση του &brandShortName;. Προειδοποίηση αν η ενημέρωση θα απενεργοποιήσει θέματα ή επεκτάσεις. Αυτή η &pref.singular; καθορίζει αν θα γίνεται προειδοποίηση πριν από την εγκατάσταση μιας ενημέρωσης που απαιτεί την απενεργοποίηση μιας ασύμβατης επέκτασης ή θέματος. Αυτή η προειδοποίηση σας επιτρέπει να αναβάλλετε την εγκατάσταση της ενημέρωσης ώστε να μην απενεργοποιηθεί κάποιο από τα πρόσθετα σας.

Αν όλα σας τα πρόσθετα είναι συμβατά ή είναι δυνατή η αναβάθμιση τους για να είναι συμβατά, η ενημέρωση του &brandShortName; θα εγκατασταθεί. Όταν γίνει επανεκκίνηση θα σας ζητηθεί να εγκαταστήσετε τις απαιτούμενες επεκτάσεις ή τις ενημερώσεις θεμάτων ώστε να συνεχίσετε να τα χρησιμοποιείτε.

Εμφάνιση ενημερώσεων
Ο &brandShortName; καταγράφει αυτόματα τις ενημερώσεις που έχετε εγκαταστήσει. Μπορείτε να δείτε πληροφορίες για αυτές τις ενημερώσεις κάνοντας κλικ σε αυτό το κουμπί.

Πρωτόκολλα

Χρήση SSL 3.0
Καθορίζει το αν θέλετε να στέλνετε και να λαμβάνετε ασφαλείς πληροφορίες μέσω SL3 (Secure Sockets Layer, Επίπεδο 3). Αυτό είναι ένα τυπικό πρωτόκολλο για ασφαλή επικοινωνία με ιστοσελίδες. Η απενεργοποίηση του θα εμποδίζει την ανάγνωση ορισμένων ιστοσελίδων.

Χρήση TLS 1.0
Καθορίζει την αποστολή και λήψη ασφαλών πληροφοριών μέσω του TLS (Transport Layer Security), ενός ανοικτού προτύπου ασφαλείας παρόμοιου με το SSL3 (Secure Sockets Layer). Η απενεργοποίηση του θα εμποδίζει την ανάγνωση ορισμένων ασφαλών ιστοσελίδων.

Επικύρωση
Το OCSP (Online Certificate Status Protocol) είναι ένας τρόπος επικύρωσης των πιστοποιητικών σας κάθε φορά που προβάλλονται ή χρησιμοποιούνται. Ο &brandShortName; δεν χρησιμοποιεί προεπιλεγμένα το OCSP, αλλά αν θέλετε να το ενεργοποιήσετε, μπορείτε να το κάνετε από εδώ. Συνήθως χρειάζεται να αλλάξετε αυτή την επιλογή μόνο αν το απαιτεί το περιβάλλον εργασίας σας.
