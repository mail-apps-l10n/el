Έλεγχος αναδυομένων

Αυτό το έγγραφο εξηγεί όλες τις διαθέσιμες &pref.plural; του &brandFullName; για τον έλεγχο των αναδυόμενων παραθύρων.

Σ' αυτή την ενότητα:
  • Τι είναι τα αναδυόμενα παράθυρα;
  • &pref.pluralCaps; φραγής αναδυομένων

Τι είναι τα αναδυόμενα παράθυρα;

Τα αναδυόμενα παράθυρα, ή πιο απλά αναδυόμενα, είναι παράθυρα που εμφανίζονται αυτόματα, χωρίς την εξουσιοδότησή σας. Έχουν διάφορα μεγέθη αλλά συνήθως δεν καταλαμβάνουν ολόκληρη την οθόνη. Μερικά αναδυόμενα ανοίγουν πάνω από το τρέχον παράθυρο του &brandShortName;, ενώ άλλα ανοίγουν στο παρασκήνιο του &brandShortName; (αναδυόμενα παρασκηνίου)

Ο &brandShortName;, σας επιτρέπει να ελέγχετε αμφότερα τα αναδυόμενα μέσω της ενότητας Περιεχόμενο στις &pref.pluralCaps;. Η φραγή αναδυομένων είναι προεπιλεγμένη, ώστε να μη χρειαστεί να ασχοληθείτε για να αποφύγετε την εμφάνισή τους στον &brandShortName;.

Όταν ο &brandShortName; αποκλείει ένα αναδυόμενο, εμφανίζει μια γραμμή πληροφοριών, και ένα εικονίδιο στη γραμμή κατάστασης. Όταν κάνετε κλικ είτε στις &pref.pluralCaps; στο κουμπί στη γραμμή πληροφοριών, είτε στο εικονίδιο στη γραμμή κατάστασης, εμφανίζεται ένα μενού με τις εξής επιλογές:

  • Να επιτρέπονται τα αναδυόμενα για αυτό τον ιστότοπο
  • Επεξεργασία Προτιμήσεων φραγής αναδυομένων
  • Να μην εμφανίζεται αυτό το μήνυμα όταν γίνεται φραγή
  • (Προβολή αναδυόμενου παράθυρου)

Η φραγή αναδυομένων ίσως να επέμβει στο περιεχόμενο μερικών δικτυακών τόπων: Μερικοί δικτυακοί τόποι, συμπεριλαμβανομένων ορισμένων σελίδων τραπεζικών συναλλαγών, χρησιμοποιούν αναδυόμενα για την υλοποίηση σημαντικών χαρακτηριστικών. Η φραγή όλων των αναδυομένων απενεργοποιεί και αυτά τα χαρακτηριστικά. Για να επιτρέψετε σε ορισμένες δικτυακές σελίδες να χρησιμοποιούν αναδυόμενα, ενώ γίνεται φραγή για όλες τις άλλες, μπορείτε να προσθέσετε συγκεκριμένες σελίδες στη λίστα επιτρεπόμενων σελίδων.

Η φραγή αναδυομένων δεν είναι πάντα αποτελεσματική: Παρόλο που ο &brandShortName; εμποδίζει την εμφάνιση των περισσότερων αναδυομένων, ορισμένες δικτυακές σελίδες, ίσως προβάλλουν αναδυόμενα χρησιμοποιώντας μεθόδους που δεν καλύπτονται από τη φραγή, ακόμα και χωρίς να είναι στη λίστα επιτρεπόμενων σελίδων.

&pref.pluralCaps; φραγής αναδυομένων

Οι &pref.pluralCaps; της φραγής αναδυομένων βρίσκονται στην ενότητα Περιεχομένο μέσω του μενού &pref.menuPath;.

Από εκεί, μπορείτε να κάνετε τα ακόλουθα:

  • Φραγή Αναδυομένων παραθύρων: Απoεπιλέξτε αυτή την &pref.singular; για να απενεργοποιήσετε τη φραγή αναδυομένων για όλες τις σελίδες.
  • Παράθυρο διαλόγου επιτρεπόμενων σελίδων : Περιέχει μια λίστα σελίδων που θέλετε να επιτρέπεται η προβολή αναδυομένων. Ο διάλογος έχει τις ακόλουθες επιλογές:
    • Να επιτρέπεται: Κάντε κλικ εδώ για να προσθέσετε μια σελίδα στη λίστα επιτρεπόμενων.
    • Απομάκρυνση σελίδας: Κάντε κλικ εδώ για να αφαιρέσετε μια σελίδα από τη λίστα επιτρεπόμενων.
    • Απομάκρυνση όλων των σελίδων: Κάντε κλικ εδώ για να αφαιρέσετε όλες τις σελίδες από τη λίστα επιτρεπόμενων.

Σημείωση: Η φραγή αναδυομένων μπορεί να μην είναι πάντα αποτελεσματική και να επέμβει στο περιεχόμενο μερικών σελίδων. Για περισσότερες πληροφορίες περί φραγής αναδυομένων, δείτε Τι είναι τα αναδυόμενα.
