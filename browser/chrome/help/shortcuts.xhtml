Συντομεύσεις πληκτρολογίου του &brandFullName;

Αυτή είναι μια λίστα των πιο κοινών συντομεύσεων πληκτρολογίου που υπάρχουν στον &brandFullName;. Αν έχετε ενεργοποιημένες τις συντομέυσεις πληκτρολογίου με στυλ Emacs στο GNOME, αυτές θα δουλεύουν και στον &brandShortName;. Όταν μια συντόμευση τύπου Emacs έρχεται σε σύγκρουση με τις προεπιλεγμένες συντομεύσεις (όπως συμβαίνει με το &accelKey;+K), θα επιλέγεται η συντόμευση τύπου Emacs αν η εστίαση είναι μέσα σε ένα πλαίσιο κειμένου (συμπεριλαμβάνεται η γραμμή διευθύνσεων και η μπάρα αναζήτησης). Σε αυτές τις περιπτώσεις θα πρέπει να χρησιμοποιείται μια εναλλακτική συντόμευση πληκτρολογίου αν κάποια είναι στη παρακάτω λίστα.
