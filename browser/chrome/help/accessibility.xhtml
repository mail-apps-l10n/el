Λειτουργίες προσιτότητας του &brandFullName;

Ο &brandShortName; περιλαμβάνει πολλά χαρακτηριστικά που κάνουν τον περιηγητή και το περιεχόμενο ιστού προσβάσιμο σε όλους τους χρήστες, ακόμα και σε αυτούς που έχουν μειωμένη ή καθόλου όραση, ή περιορισμένη δυνατότητα χρήσης πληκτρολογίου ή ποντικιού.

Σε αυτή την ενότητα:
  • Χρήση του πληκτρολογίου
  • Χρήση του ποντικιού
  • Γραμματοσειρές και χρώματα
  • Ελέγχοντας το περιεχόμενο
  • Συμβατότητα με βοηθητικές τεχνολογίες
  • Βοήθεια στο διαδίκτυο

Χρήση του πληκτρολογίου

Συντομεύσεις πληκτρολογίου

Μια λίστα με τις πιο συνηθισμένες συντομεύσεις πληκτρολογίου είναι διαθέσιμη στο Συντομεύσεις πληκτρολογίου του &brandShortName; . Μπορείτε ακόμα να βρείτε στο διαδίκτυο ένα συγκριτικό πίνακα με τις συντομεύσεις πληκτρολογίου που υπάρχουν στον &brandShortName;, τον Microsoft Internet Explorer και το Opera.

Πάντα χρήση των πλήκτρων κίνηση δρομέα για την περιήγηση μέσα στις ιστοσελίδες

Η επιλογή Πάντα χρήση των πλήκτρων κίνηση δρομέα για την περιήγηση μέσα στις ιστοσελίδες σας επιτρέπει να μετακινήστε στο περιεχόμενο της σελίδας, σαν να βρίσκεστε μέσα σε έναν επεξεργαστή κειμένου που είναι μόνο για ανάγνωση. Μπορείτε να χρησιμοποιείτε το πληκτρολόγιο για να επιλέγετε τμήματα από το περιεχόμενο της σελίδας και να το αντιγράφετε στο πρόχειρο. Μπορείτε να ενεργοποιήσετε αυτή τη λειτουργία επιλέγοντας &pref.menuPath;, και ανοίγοντας τη καρτέλα Γενικά μέσα από την ενότητα Για προχωρημένους και επιλέγοντας την Πάντα χρήση των πλήκτρων κίνηση δρομέα για την περιήγηση μέσα στις ιστοσελίδες &pref.singular;

Μπορείτε να πατήσετε F7 οποιαδήποτε στιγμή για να ενεργοποιήσετε ή να απενεργοποιήσετε αυτό το χαρακτηριστικό. Όταν πατήσετε F7 ο &brandShortName; θα σας ρωτήσει αν πράγματι θέλετε να ενεργοποιήσετε αυτό το χαρακτηριστικό. Μπορείτε να καταργήσετε την ερώτηση επιλέγοντας Να μην ξαναγίνει αυτή η ερώτηση.

Αναζήτηση για κείμενο μόλις ξεκινήσει η πληκτρολόγηση

Η επιλογή Αναζήτηση για κείμενο μόλις ξεκινήσει η πληκτρολόγηση σας επιτρέπει να μεταβείτε γρήγορα προς κάποιο κείμενο ή δεσμό σε μια ιστοσελίδα. Αυτό το χαρακτηριστικό έχει δύο λειτουργίες. Πληκτρολογήστε / και μετά το προς αναζήτηση κείμενο στην σελίδα, ή πληκτρολογήστε ' πριν το προς αναζήτηση κείμενο, αν επιθυμείτε να βρείτε μόνο δεσμούς που περιέχουν αυτό το κείμενο. Ο &brandShortName; θα τονίσει το κείμενο ή τους δεσμούς που ικανοποιούν τα κριτήρια σας καθώς πληκτρολογείτε.

Οι ακόλουθες συντομεύσεις πληκτρολογίου ελέγχουν αυτή τη λειτουργία:

  • Πληκτρολογήστε περισσοτέρους χαρακτήρες για να περιορίσετε τα αποτελέσματα της αναζήτησης.Ο &brandShortName; θα τονίσει τον επόμενο δεσμό ή το κείμενο που ταιριάζει ή θα σας ειδοποιήσει πως δεν υπάρχουν αποτελέσματα.
  • Πατήστε &accelKey;+G ή F3 για να βρείτε την επόμενη εμφάνιση του κειμένου που αναζητάτε.
  • Πιέστε &accelKey;+&shiftKey;+G ή &shiftKey;+F3 για να μεταβείτε στο προηγούμενο αποτέλεσμα της τρέχουσας αναζήτησης.
  • Πατήστε Backspace για να σβήσετε τον τελευταίο χαρακτήρα που πληκτρολογήσατε. Ο &brandShortName; θα μεταβεί στο προηγούμενο αποτέλεσμα.
  • Πατήστε Esc για να ακυρώσετε την αναζήτηση. Μπορείτε ακόμα να ακυρώσετε μια αναζήτηση, με το να αλλάξετε εστίαση να κυλίσετε τη σελίδα με το πληκτρολόγιο ή με το ποντίκι, ή απλά να περιμένετε να τερματιστεί μόνη της.

Μπορείτε να αλλάξετε τη προεπιλεγμένη συμπεριφορά αυτής της λειτουργίας επιλέγοντας &pref.menuPath;, ανοίγοντας τη καρτέλα Γενικά του πίνακα Για προχωρημένους και επιλέγοντας την &pref.singular; Αναζήτηση για κείμενο μόλις ξεκινήσει η πληκτρολόγηση Αν αυτή είναι επιλεγμένη, τότε μπορείτε να πληκτρολογείτε οποιαδήποτε στιγμή για να αρχίσει αυτόματα εύρεση στο κείμενο. Οι αναζητήσεις που ξεκινούν αυτόματα και ψάχνουν σε όλο το κείμενο στη σελίδα. Μπορείτε όμως να πατήσετε ' ή / ώστε να ξεκινήσει η εύρεση στη μια λειτουργία ή στην άλλη.

Μπορείτε να χρησιμοποιήσετε αυτή τη λειτουργία για κουμπιά, πλαίσια κειμένου και άλλα πλαίσια ελέγχου φόρμας. Αναζητήστε το κείμενο που θέλετε ακριβώς πριν το πλαίσιο ελέγχου φόρμας και πατήστε Tab. Πατώντας Tab κλείνει αυτόματα η αναζήτηση και γίνεται εστίαση στο επόμενο πεδίο φόρμας ή κουμπί, μετά από το επισημασμένο κείμενο.

Χρήση του ποντικιού

Κοινές συντομεύσεις ποντικιού

Μια λίστα των πιο κοινών συντομεύσεων ποντικιού είναι διαθέσιμη στο Συντομεύσεις ποντικιού &brandShortName;.

Γραμματοσειρές και χρώματα

Παράκαμψη γραμματοσειρών ιστοσελίδας

Μερικές ιστοσελίδες μπορεί να περιέχουν δυσανάγνωστο κείμενο ενώ άλλες μπορεί να βασίζονται για την εμφάνιση τους στις προεπιλεγμένες γραμματοσειρές του περιηγητή. Μπορείτε να καθορίσετε τις προεπιλεγμένες γραμματοσειρές σας μέσω του &pref.menuPath;, στη καρτέλα Περιεχόμενο και επιλέγοντας τη προεπιλεγμένη γραμματοσειρά και μέγεθος στην ενότητα Γραμματοσειρές & Χρώματα .

Μπορείτε να καθορίσετε την προεπιλεγμένη γραμματοσειρά για διαφορετικά στυλ, συμπεριλαμβανομένων των serif, sans-serif και monospace πατώντας το κουμπί Για προχωρημένους . Ωστόσο πολλές ιστοσελίδες καθορίζουν την δική τους γραμματοσειρά, με αποτέλεσμα οι επιλογές σας να αγνοούνται σε αυτές τις σελίδες. Για να παρακάμψετε τις γραμματοσειρές που ορίζονται απο τέτοιες σελίδες, αποεπιλέξτε τη Οι ιστοσελίδες να χρησιμοποιούν τις δικές τους γραμματοσειρές &pref.singular;

Σημειώστε πως μερικές σελίδες είναι πιθανό να προβάλλονται με λανθασμένη διάταξη, ανάλογα με την επιλογή της προεπιλεγμένης γραμματοσειράς.

Ορισμός ελάχιστου μεγέθους γραμματοσειράς

Μερικές ιστοσελίδες μπορεί να χρησιμοποιούν πολύ μικρές γραμματοσειρές στο κείμενο τους με αποτέλεσμα να είναι δυσανάγνωστες. Για να διευκολυνθείτε στην ανάγνωση τους, μπορείτε να καθορίσετε ένα ελάχιστο μέγεθος γραμματοσειράς. Έτσι οι σελίδες που θα έχουν μικρότερο από αυτό το μέγεθος κείμενο, θα εμφανίζονται από τον &brandShortName; με το μέγεθος που έχετε επιλέξει.

Για να καθορίσετε ένα ελάχιστο μέγεθος γραμματοσειράς, πηγαίνετε στο &pref.menuPath;, επιλέξτε την καρτέλα Περιεχόμενο και κάντε κλικ στο κουμπί Για προχωρημένους που βρίσκεται στην ενότητα Γραμματοσειρές & Χρώματα . Εκεί μπορείτε να καθορίσετε ένα ελάχιστο μέγεθος γραμματοσειράς από το κυλιόμενο μενού.

Σημειώστε πως μερικές σελίδες είναι πιθανό να προβάλλονται εσφαλμένα, ανάλογα με την επιλογή του ελάχιστου μεγέθους γραμματοσειράς.

Αλλαγή μεγέθους του κειμένου

Μπορείτε να αλλάξετε προσωρινά το μέγεθος του κειμένου οποιασδήποτε ιστοσελίδας από το μενού Προβολή > Μέγεθος κειμένου. Το νέο μέγεθος θα διατηρηθεί μέχρι να κλείσετε το παράθυρο ή τη καρτέλα.

Οι ακόλουθες συντομεύσεις ελέγχουν το μέγεθος του κειμένου:

  • Πατήστε &accelKey;++ για να αυξήσετε το μέγεθος του κειμένου. Γίνεται εστίαση στο κείμενο στο 120% του αρχικού του μεγέθους, έπειτα 150%, 200%, 300%, 450%, 675%, 1013%, και 1519%.
  • Πατήστε &accelKey;+- για να μειώσετε το μέγεθος του κειμένου. Αυτό μειώνει το κείμενο στο 90% του αρχικού του μεγέθους, μετά 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, και τέλος 1%
  • Πατήστε &accelKey;+0 για να επαναφέρετε το μέγεθος του κειμένου στο προεπιλεγμένο μέγεθος της σελίδας.

Σημειώστε πως μερικές σελίδες είναι πιθανό να προβάλλονται εσφαλμένα, εφόσον τροποποιήσετε το μέγεθος του κειμένου.

Παράκαμψη χρωμάτων σελίδας

Μερικές ιστοσελίδες μπορεί να χρησιμοποιούν χρώματα κείμενου παρασκηνίου με αποτέλεσμα να είναι δυσανάγνωστες, ενώ άλλες μπορεί να βασίζονται για την εμφάνιση τους στα προεπιλεγμένα χρώματα του περιηγητή. Μπορείτε να καθορίσετε τα προεπιλεγμένα σας χρώματα μέσω του &pref.menuPath;, στη καρτέλα Περιεχόμενο και κάνοντας κλικ στο Χρώματα που βρίσκεται στην ενότητα Γραμματοσειρές & Χρώματα . Μπορείτε να καθορίσετε τα προεπιλεγμένα χρώματα κειμένου και παρασκηνίου, όπως και τα χρώματα των αναγνωσμένων και μη αναγνωσμένων δεσμών. Μπορείτε ακόμα να καθορίσετε ποιοι δεσμοί θα είναι υπογραμμισμένοι.

Πολλές ιστοσελίδες όμως καθορίζουν τα δικά τους χρώματα, και έτσι οι επιλογές σας αγνοούνται. Για να παρακάμψετε τα χρώματα που καθορίζονται από τις ιστοσελίδες, αποεπιλέξτε τη Να επιτρέπεται η χρήση χρωμάτων της σελίδας αντί των επιλογών μου &pref.singular;

Χρήση ενός θέματος υψηλής αντίθεσης

Ο &brandShortName; εντοπίζει αυτόματα το αν χρησιμοποιείτε ένα θέμα υψηλής αντίθεσης, και εμφανίζει τα πάντα σε υψηλή αντίθεση. Αυτό παρακάμπτει όλες τις άλλες ρυθμίσεις του περιηγητή ή της σελίδας και επηρεάζει την επιφάνεια χρήσης του &brandShortName; (μενού, παράθυρα και πλαίσια διαλόγου) και την εμφάνιση του περιεχομένου της σελίδας που διαβάζετε.

Για να χρησιμοποιήσετε ένα θέμα υψηλής αντίθεσης, μεταβείτε στο μενού Έναρξη > Πίνακας ελέγχου > Επιλογές προσιτότητας > Εμφάνιση και επιλέξτε το Χρήση υψηλής αντίθεσης &pref.singular;

Απενεργοποίηση Java Applets

Μερικές ιστοσελίδες παρέχουν πλούσια διαδραστικότητα μέσω κάποιων Java™applets. Αυτό όμως δημιουργεί προβλήματα στους χρήστες που χρησιμοποιούν το πληκτρολόγιο για την περιήγηση τους στη σελίδα, μιας και μερικά Java applets αλλάζουν αυτόματα την εστίαση και δεν παρέχουν έναν εναλλακτικό τρόπο για έξοδο από το applet, ώστε να συνεχιστεί η περιήγηση στο υπόλοιπο της ιστοσελίδας. Αν αυτά τα applets σας δημιουργού προβλήματα, μπορείτε να απενεργοποιήσετε την Java, μέσω του μενού &pref.menuPath;, επιλέγοντας την καρτέλα Περιεχόμενο και αποεπιλέγοντας τη Ενεργοποίηση Java &pref.singular;

Έλεγχος της συμπεριφοράς της JavaScript

Μπορείτε να απενεργοποιήσετε εντελώς τη JavaScript, πηγαίνοντας στο &pref.menuPath;, επιλέγοντας Περιεχόμενο και αποεπιλέγοντας το Ενεργοποίηση JavaScript. Μερικές ιστοσελίδες που βασίζονται στη JavaScript, μπορεί να μη λειτουργούν σωστά αν η JavaScript είναι απενεργοποιημένη.

Αν δεν επιθυμείτε να απενεργοποιήσετε τη JavaScript ολοκληρωτικά, μπορείτε να χρησιμοποιήσετε τις προχωρημένες ρυθμίσεις JavaScript για να περιορίσετε τη συμπεριφορά των σεναρίων. Αμέσως μετά το πλαίσιο Ενεργοποίηση JavaScript κάντε κλικ στο κουμπί Για προχωρημένους για να ανοίξει το παράθυρο Προχωρημένες ρυθμίσεις JavaScript. Εκεί μπορείτε να ελέγχετε αν τα σενάρια ενεργειών θα μπορούν να μετακινούν ή να αλλάζουν το μέγεθος των παραθύρων, να ανασηκώνουν ή να ελαχιστοποιούν παράθυρα, να απενεργοποιούν μενού περιεχομένου, να αποκρύπτουν την γραμμή κατάστασης, ή να αλλάζουν το κείμενο της.
