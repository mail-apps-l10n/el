Για χρήστες του Internet Explorer

Αν χρησιμοποιούσατε τον Microsoft® Internet Explorer, θα βρείτε ότι είναι πολύ εύκολο να χρησιμοποιήσετε τον &brandFullName;.

Σημείωση: Αυτές οι πληροφορίες ισχύουν για το &brandFullName; για Windows και Mac OS.

Σε αυτή την ενότητα:

Τα αγαπημένα και οι ρυθμίσεις του IE

Όταν εγκαταστήσετε για πρώτη φορά τον &brandShortName;, θα εισαχθούν οι υπάρχουσες ρυθμίσεις του Internet Explorer, μαζί με τα αγαπημένα σας, τα cookies, τους αποθηκευμένους κωδικούς, και πολλά άλλα δεδομένα. Αυτό σας γλυτώνει από τον κόπο να προσαρμόσετε τον &brandShortName; στις ανάγκες σας.

Λειτουργίες &brandShortName;

  • Περιήγηση με καρτέλες: Αντί να ανοίγει ένα ξεχωριστό παράθυρο για κάθε σελίδα που θέλετε να δείτε, μπορείτε να ανοίγετε πολλές σελίδες στο ίδιο παράθυρο με καρτέλες. Μπορείτε ακόμα να ορίσετε μια ομάδα από καρτέλες ως την αρχική σας σελίδα. Για περισσότερες πληροφορίες δείτε το Περιήγηση με καρτέλες.
  • Έλεγχοι αναδυόμενων παραθύρων: Σας επιτρέπει να κάνετε φραγή αναδυόμενων παραθύρων.
  • Διαχείριση Cookies: Σας επιτρέπει να αλλάξετε το τι θα κάνει ο &brandShortName; όταν δέχεται cookies.
  • Διαχείριση λήψεων: Οργανώνει τις λήψεις των αρχείων αποθηκεύοντας τις σε ένα απλό παράθυρο για να ελαχιστοποιηθεί ο συνωστισμός. Μπορείτε να έχετε πλήρη πρόσβαση στα στατιστικά λήψης χωρίς να χρειάζεται να χρησιμοποιείτε πολλαπλά παράθυρα για κάθε λήψη.
  • Διαχείριση εικόνων: Σας επιτρέπει να απενεργοποιείτε την εμφάνιση εικόνων από συγκεκριμένες ιστοσελίδες ή να τις απενεργοποιήσετε εντελώς. Αυτό θα σας φανεί χρήσιμο αν θέλετε να μειώσετε το χρόνο φόρτωσης των ιστοσελίδων.

Συντομεύσεις πληκτρολογίου

Θα δείτε ότι ο Microsoft Internet Explorer και ο &brandShortName; χρησιμοποιούν πολλά ίδια πλήκτρα συντόμευσης. Για μια πλήρη λίστα πλήκτρων συντόμευσης δείτε το Λίστα συντομεύσεων πληκτρολογίου του &brandShortName;.
